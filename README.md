# hanyupy (hanyupy)

让你可以用中文编写Python代码的库！

## 安装

```bash
pip install hanyupy
```

## 使用方法

只需要导入hanpy，就可以用中文编程了！

```python
import hanpy

# 基础输出
打印("你好，世界！")

# 变量和输入
姓名 = 输入("请输入你的名字：")
打印("你好，" + 姓名)

# 数学运算
结果 = 加(5, 3)
打印(结果)  # 输出: 8

# 条件判断
如果 年龄 大于等于 18：
    打印("成年人")
否则：
    打印("未成年人")

# 循环
循环 数字 在 范围(5)：
    打印(数字)

# 列表操作
水果列表 = 列表(["苹果", "香蕉", "橙子"])
水果列表.添加("葡萄")
打印(水果列表)

# 字典操作
个人信息 = 字典({"姓名": "张三", "年龄": 25})
打印(个人信息.获取("姓名"))

# 数学函数
导入 数学
打印(数学.平方根(16))  # 输出: 4.0
打印(数学.圆周率)      # 输出: 3.14159...

# 随机数
导入 随机
打印(随机.随机整数(1, 100))
```

## 完整示例

```python
import hanpy

# 计算器示例
定义 计算器():
    打印("简单计算器")
    
    数字1 = 浮点数(输入("请输入第一个数字："))
    运算符 = 输入("请输入运算符（+、-、*、/）：")
    数字2 = 浮点数(输入("请输入第二个数字："))
    
    如果 运算符 等于 "+":
        结果 = 数字1 + 数字2
    否则如果 运算符 等于 "-":
        结果 = 数字1 - 数字2
    否则如果 运算符 等于 "*":
        结果 = 数字1 * 数字2
    否则如果 运算符 等于 "/":
        结果 = 数字1 / 数字2
    否则：
        打印("无效的运算符")
        返回
    
    打印("结果是：" + 字符串(结果))

# 猜数字游戏
定义 猜数字游戏():
    导入 随机
    答案 = 随机.随机整数(1, 100)
    猜测次数 = 0
    
    打印("猜数字游戏！我已经想好了一个1到100之间的数字。")
    
    循环：
        猜测 = 整数(输入("请输入你的猜测："))
        猜测次数 = 猜测次数 + 1
        
        如果 猜测 小于 答案：
            打印("太小了！")
        否则如果 猜测 大于 答案：
            打印("太大了！")
        否则：
            打印("恭喜你猜对了！用了" + 字符串(猜测次数) + "次。")
            中断

# 学生成绩管理系统
定义 成绩管理系统():
    学生列表 = 列表()
    
    循环：
        打印("\n1. 添加学生")
        打印("2. 查看所有学生")
        打印("3. 退出")
        选择 = 输入("请选择：")
        
        如果 选择 等于 "1":
            姓名 = 输入("请输入学生姓名：")
            成绩 = 浮点数(输入("请输入学生成绩："))
            学生列表.添加({"姓名": 姓名, "成绩": 成绩})
            打印("添加成功！")
        
        否则如果 选择 等于 "2":
            打印("\n学生列表：")
            循环 学生 在 学生列表：
                打印("姓名：" + 学生.获取("姓名") + "，成绩：" + 字符串(学生.获取("成绩")))
        
        否则如果 选择 等于 "3":
            打印("再见！")
            中断
        
        否则：
            打印("无效的选择！")
```

## 注意事项

1. hanpy 会在导入时修改 Python 的 builtins，因此只需要导入一次
2. 某些高级特性可能仍然需要使用英文关键字
3. 建议在学习和演示中使用，生产环境请谨慎使用
